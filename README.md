# Roulette Night 18+

Hotová jednoduchá PWA aplikace pro GitHub Pages.

## Jak to nahrát na GitHub
1. Rozbal ZIP.
2. Nahraj všechny soubory do rootu repozitáře.
3. V GitHubu otevři **Settings → Pages**.
4. U **Source** vyber větev, typicky **main**, a složku **/(root)**.
5. Ulož.
6. Po chvilce bude appka běžet na tvé GitHub Pages URL.

- sexy dark vzhled
- režimy: Vše / Flirt / Dotek / Hra
- náhodné výzvy
- historie kol
- základní statistiky
- PWA instalace na plochu
